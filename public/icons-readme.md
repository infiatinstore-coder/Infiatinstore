# PWA Icons

Place your PWA icons here:

- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels

These icons will be used for:
- App installation on mobile devices
- Home screen shortcuts
- Splash screens

## Generate Icons

You can use these tools to generate PWA icons:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/

## Design Guidelines

- Use your brand colors (#667eea - primary)
- Keep it simple and recognizable
- Should work well in both light and dark modes
- Include your logo or brand initial
